Intermédiaire de paiement Monetico

TPE virtuel distribué par le CIC et le Crédit Mutuel

Description

Ce module est un intérmediaire de paiement qui utilise la solution Monetico Paiement. Il permet de proposer un mode de paiement standard ainsi qu'un mode de paiement en plusieurs fois

Usage

Une fois le module installé, rendez-vous dans le menu Site Web > Configuration > Intérmédiaires de paiement
